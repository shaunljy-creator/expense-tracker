# Personal Expense Tracker (Python)

A simple personal finance tracker built in Python using:
- SQLite (data storage)
- Modular backend architecture
- CRUD operations
- Week 1 of Python Winter Break Project

## Project Structure

- `models/` — data classes
- `services/` — database + business logic
- `data/` — SQLite database file
- `main.py` — runnable entry point

## How to Run
```bash
python main.py
